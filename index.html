Data Science Portfolio

My name is Kevin Ferreira, I graduated from Florida International University (FIU) in 2023 with a B.S. Degree in Mathematics (Statistics track). I conducted a regression analysis on depression rates in 40 different countries under the guidance of Dr. Golam Kibria from the Department of Mathematics and Statistics at FIU. The report for this project presents already processed data after a regression analysis using R. An example of the code in R can be seen here. To showcase fine tuned skills learned in college or newly learned skills, I reproduced using Python the same results obtained using R. Furthermore, I used SQL to clean and organize data to show trends and relations that may escape a first view of the data, and for data visualization I used Excel and Tableau. The raw data was collected from the following websites: WorldPopulationReview, OurWorldinData, Worldbank, Worldometers, and GHsindex.
